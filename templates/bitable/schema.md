# 飞书多维表字段模板

公开模板名建议：

```text
Lark Reading Calendar 阅读收件箱
```

## 数据表：阅读收件箱

| 字段名 | 类型 | 是否必需 | 说明 |
| --- | --- | --- | --- |
| 内容 | 多行文本 | 是 | 原始收集文本 |
| 标题 | 文本 | 否 | 链接标题识别结果 |
| 链接 | URL | 否 | 内容里的第一条链接 |
| 平台 | 单选 | 否 | wechat / bilibili / xiaohongshu / web / unknown |
| 来源 | 单选 | 否 | iOS 快捷指令 / OpenClaw / 手工测试 |
| 通道 | 单选 | 否 | wechat / test / other |
| 入口 | 文本 | 否 | wechat-clipboard / openclaw-wechat / curl |
| 发送人 | 文本 | 否 | 机器人入口发送人 |
| 消息ID | 文本 | 否 | 去重 ID |
| CloudBaseItemId | 文本 | 是 | CloudBase item `_id` |
| 状态 | 单选 | 是 | 待读 / 已排期 / 已读完 / 未读完 / 稍后再读 / 放弃 |
| 收集时间 | 日期 | 是 | 入队时间 |
| 批次ID | 文本 | 否 | 所属批次 ID |
| 日历开始 | 日期 | 否 | 阅读 block 开始时间 |
| 日历结束 | 日期 | 否 | 阅读 block 结束时间 |
| 日历链接 | URL | 否 | 飞书日历事件链接 |
| 反馈时间 | 日期 | 否 | 用户提交反馈的时间 |
| 阅读反馈 | 单选 | 否 | 已读完 / 未读完 / 稍后再读 / 放弃 |
| 备注 | 多行文本 | 否 | 用户反馈备注 |

## 推荐视图

### 待读

筛选：

```text
状态 = 待读
```

排序：

```text
收集时间 升序
```

### 已排期

筛选：

```text
状态 = 已排期
```

排序：

```text
日历开始 升序
```

### 已完成

筛选：

```text
阅读反馈 不为空
```

排序：

```text
反馈时间 倒序
```

## 公开模板发布检查

- 模板中不要包含真实阅读记录。
- 模板中不要包含真实 app token、table id、CloudBase URL。
- 模板内云文档教程使用占位符。
- 分享权限设置为允许复制。
